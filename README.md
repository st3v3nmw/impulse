# impulse

Distributed Key Value Store in Go

## Features

### v0 (Basic Interface on a Single Machine)

- [x] HTTP Server
- [x] `Put(key, value)`
- [x] `Get(key)`
- [x] `Delete(key)`

### v1 (Master-Slave Architecture)

- [ ] Replicas
- [ ] Chaos Testing

### v2 (Sharding)

- [ ] Sharding

### v3 (Complex Querying)

- [ ] Range Queries
- [ ] Search

### v4 (No Leader Nodes)

- [ ] [Raft Consensus](https://raft.github.io/)
